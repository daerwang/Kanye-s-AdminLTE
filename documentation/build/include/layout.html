布局

布局主要由四部分组成

  • Wrapper .wrapper. 一个将整张页面包装的div
  • Main Header .main-header. 包括Logo和导航条
  • Sidebar .sidebar-wrapper.包括用户面板和菜单
  • Content.content-wrapper. 包括页面标题和内容

提示!

如果你是从头开始做网站,请从starter 页面开始

布局选项

AdminLTE 2.0为你的布局提供了一些选项.每种样式都可以被添加到body上以达到设计目标.

  • 自适应布局: 使用样式.fixed来获得一个自适应的页头和工具栏.
  • 折叠工具栏: 使用样式 .sidebar-collapse 来将工具栏折叠起来
  • 盒式布局: 使用样式 .layout-boxed 会使一个不论浏览器宽度为多少Wrapper都只有1250px.
  • 顶部导航: 使用样式 .layout-top-nav 会移除 sidebar ,将所有的链接放在页面顶部.

Note: 你无法同时使用盒式布局和自适应布局.其他布局可以一起使用.
